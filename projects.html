NAVI — Vision Restored

Overview: NAVI is an AI-powered wearable and mobile solution created for the Microsoft Imagine Cup & Datathon 2024. It empowers blind and visually impaired individuals in Kenya through real-time object recognition, scene narration, and natural language interaction using Microsoft Azure Cognitive Services.

✓ Successfully passed Stage 1 of the Microsoft Imagine Cup 2024.

Features
  • Real-Time Object Recognition — Azure AI Vision detects everyday items in the user's environment
  • Scene Understanding — Narrates full scenes using Computer Vision APIs for spatial awareness
  • Text-to-Speech — Reads signs, books, and objects aloud with natural voice options
  • Speech-to-Text — Converts user voice commands into text for hands-free control
  • 10-Language NLP — Custom datasets for 10 African languages including Swahili and English
Social Impact

Designed to assist 800,000+ visually impaired individuals aged 15–35 in Kenya. Co-designed with blind students at USIU-Africa to ensure real-world usability.

Tech Stack
Android JavaXML Microsoft Azure AI VisionAzure Speech Services NLPWeb API Integration
LUMEA — Booking & Payment System

Overview: LUMEA is an end-to-end booking and M-Pesa payment platform currently in active development at Quantra Core Limited. Built with a Flutter mobile frontend and Cloudflare Workers serverless backend.

Key Features
  • Flutter mobile UI with smooth booking flow and real-time state management
  • M-Pesa STK Push integration for frictionless mobile payments
  • Firebase Auth with secure Firestore rules for user data protection
  • Cloudflare Worker REST APIs — edge-deployed for low latency
  • Full production documentation and deployment strategy
