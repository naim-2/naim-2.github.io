Sentiment Analysis Web App

Overview: The Sentiment Analysis Web App allows users to input text and get a sentiment analysis of the messages. It classifies messages as positive, negative, or neutral, providing a score for each message.

Sentiment Analysis Web App Screenshot

  • Text Sentiment Analysis: Classifies text messages into positive, negative, or neutral sentiments.
  • Real-time Processing: Provides instant feedback as soon as the user submits the input.
  • Statistics: Displays overall sentiment statistics (positive, neutral, negative) for all messages.
  • Rate Limiting: Prevents spamming by limiting the number of requests from each user.

The interface is designed to be clean and simple, with a focus on ease of use. Users can enter multiple messages, analyze them, and view the results instantly.

The backend is powered by Flask and uses TextBlob for sentiment analysis. The API limits the rate of requests to ensure fairness for all users.

Personal Journaling App

Overview: The Personal Journaling App is a mobile application designed to provide users with a platform to document their thoughts and experiences. Built with React Native and TypeScript, this application allows for secure user authentication, comprehensive journal entry management, and insightful data summarization.

Personal Journaling App Demo Video

  • User Authentication: Users can sign up and log in securely using JWT or session-based authentication.
  • Journal Entry Management: Create, read, update, and delete journal entries with fields for title, content, category, and date.
  • Categorization: Organize entries into categories such as Personal, Work, and Travel for easy navigation.
  • Summary View: Generate summaries of entries over specified periods (daily, weekly, monthly) for reflective insights.
  • Settings: Users can update their usernames and passwords to manage their profiles.

The application features a simple, intuitive design focusing on usability, utilizing pre-designed UI libraries to enhance user experience.

The backend is built using Express.js with TypeScript, providing secure user management and CRUD operations for journal entries.

Utilizes a relational database (a hosted PostgreSQL) for efficient data management.
