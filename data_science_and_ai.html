Cyber Shujaa — Data Science & AI Track
8-week intensive programme · Sept – Nov 2025

A structured programme covering the full data science pipeline — from web scraping and data wrangling through to advanced classification models and business intelligence dashboards. Each week built toward real-world competency with Python, scikit-learn, Tableau, and Power BI.

8 Week
10/11/2025
Classification Models with Python

Built and evaluated six classification models on the Wine dataset — Logistic Regression, Decision Tree, Random Forest, KNN, Naive Bayes, and SVM. Random Forest and SVM achieved the highest accuracy.

  • EDA with correlation heatmaps and feature visualizations
  • Feature standardization for SVM and KNN
  • Confusion matrix visualization for all six models
  • Model comparison using accuracy scores and classification reports
View on Google Colab
7 Week
03/11/2025
Regression Models with Python

Simple and multiple linear regression to predict house prices by area, bedrooms, and age. Achieved R² of 95.5% on the multiple regression model.

  • Loaded and explored housing dataset in Google Colab
  • Handled missing data by imputing median values
  • Evaluated with MAE, MSE, RMSE, and R² metrics
  • Visualized actual vs predicted prices with regression lines
View on Google Colab
5 Week
20/10/2025
Data Visualization with Tableau

First experience with Tableau — loaded, transformed, and visualized HR employee datasets. Built a comprehensive dark-themed dashboard with Overview, Demographics, and Income Analysis sections.

  • Created calculated measures: Total Hired, Terminated, Active
  • Built bar charts, line charts, area charts, pie charts, and scatter plots
  • Applied consistent color schemes for professional dark layout
  • Published and shared as portfolio project
View on Google Drive
4 Week
13/10/2025
Business Intelligence with Power BI

First Power BI experience — loaded hotel business data from multiple Excel and CSV files, built DAX measures, and published an interactive revenue and bookings dashboard.

  • Managed data relationships and created calculated columns with DAX
  • Built measures for revenue, bookings, and capacity
  • Designed interactive dashboards with filters and visualizations
  • Published report to Power BI Service
View on Google Drive

Titanic Dataset Exploratory Data Analysis

EDA on the Titanic dataset using Kaggle. Identified survival factors through univariate, bivariate, and multivariate analysis with correlation heatmaps and histograms.

  • Handled missing values and removed irrelevant columns
  • Conducted survival factor analysis: gender, class, age
  • Generated correlation heatmaps and bar charts
View on Kaggle

Data wrangling on the Netflix shows/movies dataset — cleaning, transformation, validation, and export to clean CSV using pandas.

  • Fixed missing directors, countries, and date formats

  • Created new features like year_added
  • Exported and shared the cleaned dataset
