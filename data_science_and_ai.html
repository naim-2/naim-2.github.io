Cyber Shujaa — Data Science & AI Track
12-week intensive programme · Sept – Dec 2025

A structured programme covering the full data science and AI pipeline — from web scraping and data wrangling to machine learning, deep learning, natural language processing, and generative AI. Each week built toward real-world competency through hands-on projects using Python, scikit-learn, TensorFlow/Keras, Hugging Face Transformers, LangChain, Tableau, and Power BI.

12 Week
03/12/2025
Generative AI & RAG Pipeline

Built a Retrieval-Augmented Generation (RAG) system using LangChain, FAISS, and Hugging Face models to answer questions from PDF documents. Improved answer accuracy using context-aware retrieval.

  • Extracted and chunked text from PDF documents
  • Generated embeddings using Sentence-Transformers
  • Stored vectors in FAISS for efficient similarity search
  • Implemented retrieval-based Q&A pipeline with LangChain
  • Integrated Hugging Face LLM for answer generation
  • Compared RAG vs non-RAG responses for accuracy
  • Applied prompt engineering for better response quality
View on Google Colab
11 Week
27/11/2025
NLP with Transformers (BERT Embeddings)

Applied a pre-trained BERT model using Hugging Face Transformers to generate contextual embeddings and evaluate sentence similarity. Achieved ~80% accuracy using cosine similarity.

  • Loaded bert-base-uncased tokenizer and model
  • Generated contextual embeddings for sentence pairs
  • Demonstrated polysemy using context-aware representations
  • Computed cosine similarity between sentence embeddings
  • Evaluated predictions against labeled similarity data
  • Explained differences between BERT and traditional NLP (TF-IDF, BoW)
  • Used [CLS] token for sentence-level representation
View on Google Colab

Deep Learning with ANN (MNIST Classification)

Built and trained an Artificial Neural Network (ANN) using Keras to classify handwritten digits from the MNIST dataset. Achieved high accuracy on unseen test data.

  • Preprocessed image data (normalization and one-hot encoding)
  • Designed ANN architecture using Keras Sequential API
  • Model training with Adam optimizer and validation tracking
  • Evaluated performance using accuracy, loss, and test dataset
  • Visualized training history (loss & accuracy curves)
  • Generated confusion matrix and classification report
  • Saved and reloaded model using modern Keras format
View on Google Colab

Built an end-to-end machine learning pipeline using the California Housing dataset, including preprocessing, hyperparameter tuning with GridSearchCV, and deployment using FastAPI. Delivered a fully functional /predict API endpoint.

  • Data preprocessing with ColumnTransformer and StandardScaler
  • Pipeline creation with KNeighborsRegressor
  • Hyperparameter tuning using GridSearchCV (5-fold CV)
  • Model evaluation using R² score

  • Built and tested FastAPI /predict endpoint for real-time predictions
